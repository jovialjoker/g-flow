# G-Flow

### Ce este G-Flow?

G-Flow este o unealta vizuala colaborativa care ajuta la transfomarea modului in care echipele lucreaza impreuna

### Cum te autentifici?

Platforma G-Flow pune la dispozitie o autentificare de tip third party prin intermediul GitHub API ce asigura unele elemente de securitate in ceea ce priveste salvarea conturilor in baza de date

### Cum functioneaza?

Aplicatia este formata din 2 module: 
* **Modulul de liste** *prin care utilizatorul poate crea sau poate fi invitat intr-o lista de task-uri*
* **Modulul de GitHub** *prin care utilizatorul poate modifica in mod direct repository-ul in functie de ce ii cere task-ul*

### Module auxiliare:
* *Modulul de dashboard*
* *Modulul de cautare*
* *Modulul de schimbare de limba*
* *Modulul pentru vizualizarea profilului*

### Tehnologii 

* Bootstrap 4
* CodeIgniter 4
* MySQL
* Node modules 
* PHP dependencies ( manager: composer)
